## 🌟 Features

* 🎯 Multiple categories (Tech / General / Aptitude)
* 🧠 30+ questions with random shuffle
* ✅ Correct (Green) / ❌ Wrong (Red) feedback
* 🎬 Animations (shake, glow, score pop)
* 🔊 Sound effects
* 🎉 Confetti on completion
* 👤 Avatar + Profile system
* 📱 Mobile-style bottom navigation
* 🌌 3D animated background
